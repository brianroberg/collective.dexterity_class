collective.dexterity_class Installation
---------------------------------------

To install collective.dexterity_class using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.dexterity_class`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.dexterity_class
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
